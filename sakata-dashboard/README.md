# 사카다 5법 — KOSPI · KOSDAQ 단계 진단 대시보드

> 🌐 라이브 데모: https://jihansong.github.io/team-market/



혼마 무네히사(本間 宗久, 1724~1803)의 **사카다 5법**(三山·三川·三空·三兵·三法)을
KOSPI / KOSDAQ 일봉에 자동 적용해, 현재 시황이 어떤 단계
(반전 시도 / 추세 소진 / 추세 지속 / 추세 재개 / 횡보)에 있는지 분류하는 정적 웹 대시보드.

## 실행

서버 없이 정적 파일만으로 동작합니다. 단, `file://` 로 열면 CORS로 데이터 fetch가 실패할 수 있어 로컬 HTTP 서버를 권장합니다.

```bash
# Python (대부분 설치되어 있음)
cd sakata-dashboard
python3 -m http.server 8080
# 브라우저에서 http://localhost:8080 열기
```

또는 VSCode의 Live Server 확장으로 `index.html` 우클릭 → "Open with Live Server".

## 데이터 소스

- [Stooq](https://stooq.com) 일봉 CSV (`^kspc`, `^kosdaq`)
- 무료, 무인증, CORS 허용. 종가 기준 일봉만 사용.

## 사카다 5법 감지 규칙 (휴리스틱)

| 법 | 규칙 (요약) |
|---|---|
| **三山** | 최근 120봉 내 fractal 고점 3개의 가격 편차가 ≤ 4%면 검출 |
| **三川** | 최근 120봉 내 fractal 저점 3개의 가격 편차가 ≤ 4%면 검출 |
| **三空** | 같은 방향 갭(전일 `high`↔당일 `low` 또는 반대)이 3거래일 연속 |
| **三兵** | 같은 색 캔들 3연속 + 종가 점진 진행 + 몸통이 20봉 평균 range의 30% 이상 |
| **三法** | 추세봉(몸통 ≥ 20봉 평균 range의 80%) + 반대색 작은 봉 3개(추세봉 range 내부) + 추세봉 종가 돌파 봉 |

표준 정의가 없으므로 임계값은 한국 지수 변동성을 고려한 휴리스틱입니다. `sakata.js` 상단 상수만 바꾸면 민감도 조절 가능.

## 단계 분류 우선순위

최근 5거래일 내 감지된 신호 중 우선순위 적용:

1. **三空** → "추세 소진 임박" (warning)
2. **三山** → "약세 반전 시도"
3. **三川** → "강세 반전 시도"
4. **三法** → "추세 재개" (방향에 따라 강/약세)
5. **三兵** → "추세 지속" (적/흑삼병)
6. 그 외: 20/60일 수익률로 약상승/약하락/횡보 분류

## 기술 레벨 (차트 수평선)

- 60일 신고가 (돌파선, 실선 녹)
- 최근 5일 저점 평균 (단기 목선, 점선 황)
- Fibonacci 38.2% / 61.8% 되돌림 (60일 swing 기준)
- 60일 / 120일 이동평균선
