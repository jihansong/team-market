# 레퍼런스 라우팅 가이드

## 트리거 어휘

사용자가 매거진 / 뉴스 / 리포트 / 영상 / 콘텐츠 제작을 요청하면서 아래 키워드 중 하나라도 사용하면, 이 문서의 "레퍼런스 사이트 7선"을 참고 출처로 활용한다.

- 감도 높은 / 감도 있는
- 느낌 있는
- 섹시한
- 느좋 / 느좋인
- 세련된
- (그 외 유사 키워드: 무드 있는, 트렌디한, 에디토리얼한, hif스러운)

## 응답 원칙

위 트리거가 감지되면 다음을 지킨다.

1. **"어디서 참고했는지" 명시한다.** 결과물(카피·레이아웃·컬러팔레트·룩북 등)을 제시할 때, 아래 7개 중 어떤 사이트의 어떤 카테고리·키워드 검색을 근거로 했는지 함께 적는다. 예: "tagwalk에서 `red shoes` × FW25 검색", "pantone 15-6324 TCX(Peapod) 기준 그린 톤".
2. **구체적 코드·URL을 인용한다.** Pantone은 컬러 코드, tagwalk는 검색 쿼리, are.na는 채널명, steep.design은 카테고리(Fashion / Typography 등)를 명시한다. 모호한 "감각적인 톤" 같은 표현은 금지.
3. **자체 데이터 환각 금지.** 사이트 내부 콘텐츠를 직접 확인하지 않은 상태라면, "tvcf 메인에 X 광고가 있다" 같은 추측성 인용은 하지 않는다. 대신 "tvcf의 `오늘의 크리에이티브` 섹션에서 [브랜드/카테고리] 키워드로 직접 확인 권장" 식으로 안내한다.
4. **여러 사이트를 조합한다.** 예: 컬러(pantone) × 룩(tagwalk) × 무드보드(are.na) × 광고 카피(tvcf). 단일 사이트만 인용하지 않는다.
5. **결과물에 hif 스타일을 입힌다.** hif.community의 매거진 어투(부드러운 1인칭, 짧은 호흡, 이모티콘 절제, "~예요/~네요" 어미)를 기본 톤으로 한다.

## 레퍼런스 사이트 7선

### 1. TVCF — 광고 아카이브 플랫폼
- URL: https://tvcf.co.kr/
- 용도: TV CF, 숏폼, 해외 광고까지 영상·이미지·제작사 레퍼런스 통합 검색. 광고 카피·편집 스타일·색감·브랜드 감성 트렌드.
- 활용 트리거: "광고 톤의 카피", "CF 같은 영상 컨셉", "브랜드 무드"
- 검색 팁: `오늘의 크리에이티브` 섹션이 일일 큐레이션, `New Creative` 라벨이 최신 트렌드.

### 2. snskeyboard — 이모티콘 / 특수문자
- URL: https://snskeyboard.com
- 용도: SNS용 이모티콘, 특수문자, 팬시 폰트, 아스키 아트, 도트 아트, 시끄러운 댓글. 키치·레트로·귀여운 무드.
- 활용 트리거: "프로필 꾸미기", "키치한 텍스트", "이모티콘 깔린 카피"
- 카테고리: `데코 폰트`, `팬시 폰트`, `이모티콘 메이커`, `아스키 캘린더`, `상형 문자`, `그 시절 글씨체` 등.

### 3. Pantone — 컬러칩 (공식)
- URL: https://www.pantone.com/color-finder
- 용도: 팬톤 공식 컬러 코드 검색. 콘텐츠의 키 컬러 레퍼런스 잡을 때 신뢰도 가장 높음.
- 활용 트리거: "키 컬러 잡아줘", "이 무드의 팬톤 코드", "컬러 팔레트"
- 카테고리 (FASHION, HOME + INTERIORS): COTTON TCX, NYLON BRIGHTS TN, METALLIC SHIMMERS TPM, PAPER TPG, POLYESTER TSX, SKINTONE GUIDE / (GRAPHICS): CMYK COATED/UNCOATED, COLOR BRIDGE COATED.
- **항상 컬러 코드 (예: `15-6324 TCX Peapod`)와 카테고리를 함께 인용한다.**

### 4. steep.design — 인스타그램 스토리 레퍼런스
- URL: https://steep.design
- 용도: 카테고리별 IG 스토리 디자인 아카이브. 패션·뷰티·F&B 등 해외 브랜드 스토리.
- 활용 트리거: "인스타 스토리 레이아웃", "브랜드 스토리 디자인", "9:16 콘텐츠"
- 카테고리 예: Fashion (wax, Organic Basics, Dior, COS), Typography (KEA, Spotify, Twitch, Uber).

### 5. cara — 아트 포트폴리오
- URL: https://cara.app/explore
- 용도: 일러스트레이터·디자이너용 아트 SNS / 포트폴리오. 그림, 3D, 컨셉아트. **AI 생성 이미지 제한 정책**으로 사람 아티스트의 비주얼만 큐레이션.
- 활용 트리거: "오리지널 일러스트 톤", "AI 안 들어간 아트워크", "컨셉아트 레퍼런스"
- 섹션: Discover / Following / Latest, 이벤트 챌린지(#Plantober, #Huevember 등) 활발.

### 6. are.na — 취향 기반 아카이빙
- URL: https://www.are.na/editorial
- 용도: 이미지+글+링크+영상 무드보드. 핀터레스트와 달리 **알고리즘·광고·좋아요 없이** 영감 아카이빙에만 집중.
- 활용 트리거: "무드보드", "에디토리얼 아카이브", "조용한 영감"
- 채널 예: `Editorial`, `styling/art direction`, `Featured Channels`, `Direction / Editorials / General style`, `AD – On Location`, `On Body`.

### 7. tagwalk — 패션 런웨이 검색
- URL: https://www.tag-walk.com/en/
- 용도: 패션계의 구글. 세계 패션쇼 런웨이 룩 검색. 12000+ collections, 1400+ designers, 50+ seasons.
- 활용 트리거: "런웨이 룩 레퍼런스", "이 시즌 트렌드", "디자이너 룩북"
- 검색 차원: **브랜드 × 시즌 × 컬러 × 아이템**(예: `red shoes`). 미니멀·에디토리얼·럭셔리 무드에 강함.

## 결합 시나리오 (예시)

요청: "감도 있는 뷰티 매거진 컨셉 잡아줘"
→
- **컬러**: pantone에서 키 컬러 2-3개 (예: `15-6322 TCX Light Grass Green`)
- **룩 레퍼런스**: tagwalk에서 `green` × `beauty` 시즌 검색 결과 인용
- **레이아웃 무드**: are.na `Editorial` 채널 + steep.design Fashion 카테고리
- **카피 톤**: tvcf 뷰티 광고 (예: 닥터지·이니스프리 최근 CF) 카피 스타일 차용
- **장식 요소**: snskeyboard에서 키치 아닌 미니멀 특수문자 (필요 시)
- **비주얼 컨셉아트**: cara에서 일러스트 무드 (필요 시)

## 환경 메모

이 레퍼런스 라우팅은 사이트 콘텐츠를 실시간 크롤링해서 활용하는 것을 전제로 한다. 단, **Claude Code on the web의 "GitHub-only" 네트워크 정책 환경에서는 이 사이트들에 직접 접속할 수 없다.** 실시간 크롤링/스크린샷이 필요한 작업은 로컬 PC의 Claude Code(Playwright MCP 포함 `seren-tools` 플러그인)에서 진행한다. 웹 환경에서는 사용자에게 어느 사이트의 어느 검색을 직접 열어보라고 안내하는 식으로 우회한다.

## 전역 적용

이 파일은 `team-market` 레포 안에서 작업할 때만 자동 로드된다. 모든 프로젝트에서 위 라우팅을 쓰고 싶으면 이 파일 내용을 `~/.claude/CLAUDE.md`(사용자 전역)로 복사하거나, `seren-tools` 플러그인의 skill로 래핑한다.
