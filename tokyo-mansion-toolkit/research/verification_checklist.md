# 매물 검증 체크리스트 (도쿄 핵심지구 1LDK+)

> 후보 빌딩의 매도 호실을 SUUMO/HOMES 등에서 식별한 뒤, 본 체크리스트로 항목별 검증을 통과한 매물만 최종 10건 풀에 올릴 것.  
> 본 체크리스트는 **자기자본 KRW 450M / LTV 60-75% / DSCR ≥ 1.0x** 조건의 실투자 결정용입니다.

---

## A. 매물 기본 사실 검증

- [ ] 현재 매도 중인 호실인가? (SUUMO·HOMES·athome 3개 사이트 교차확인)
- [ ] 매도 호실의 동·층·向 일치 (서로 다른 사이트가 같은 호실인지 확인)
- [ ] 호가 표기 통일 (税込 vs 税抜, 만엔 단위)
- [ ] 마지막 가격 변경일 + 등록 경과 일수 (90일+ 이상은 가격 협상 여지)
- [ ] 매도자 신분 (個人 / 法人 / リフォーム済 업자)

## B. 빌딩 단지 검증

- [ ] 분양시점·시공사·관리회사 (`research/tokyo_core_buildings.json` 대조)
- [ ] 총세대수 (300세대+ 권장 = 임대시장 가격 형성 용이)
- [ ] 신내진(1981년 이후 신축) 또는 旧耐震 보강 이력
- [ ] 분양 직후 ~ 5년 차의 단지인가 (수선적립금 부담 큰 시점인가)
- [ ] 大規模修繕 시기 (앞으로 3~5년 내 일시금 부담 예정인가)
- [ ] 駐車場 비율 (도쿄 도심은 거의 사용 안 함 but 카탈로그 항목)
- [ ] 戸数 대비 賃貸비율 (40%+면 임대 위주 단지)

## C. 호실 사실 검증

- [ ] 専有面積 (壁芯·内法 구분), 발코니 면적, 향 (남·동남이 가격 프리미엄)
- [ ] 築年月·階·간토리·실내 리노베이션 이력
- [ ] 관리비 + 수선적립금 + 기타 月額費 합계
- [ ] 固定資産税·都市計画税 직전 년도 부과액 (관리실 또는 매도자 자료 요청)
- [ ] 임대시 想定 賃料 (같은 단지 내 임대 매물 OR 관리회사 가이드)
- [ ] 외부 임대시장 시세 (HOMES 賃貸 또는 SUUMO 賃貸로 같은 단지 평형 비교 3건+)

## D. 재무 검증 (`src/financial_model.py` 사용)

- [ ] **DSCR ≥ 1.0x** at LTV 70% / 2.5% / 30년 amort.
- [ ] DSCR 민감도: 임대료 -10% 시 0.9x 이상인지 stress test
- [ ] 가용 자기자본 한도 안에 들어오는지 (`equity_required` ≤ ¥47.4M)
- [ ] Cash-on-Cash ≥ 0% (마이너스면 carry-cost 매년 발생)
- [ ] 표면이율 ≥ 3.5% (도쿄 도심 1LDK 기준선)
- [ ] 環境 stress test: 금리 +1% (3.5%) 시 DSCR 시뮬레이션

## E. 매수자(차주) 자격 검증

- [ ] 사용자가 일본 거주자(영주권·배우자·취업비자)인지
- [ ] 비거주자라면 어떤 은행이 대출 가능한가 (SBI신생/東京스타/SMBC프레스티지/오릭스은행)
- [ ] 비거주자 대출시 LTV 한도 (보통 50-60%) 와 금리 (2.5-4.5%)
- [ ] 円화 대출 vs 한화/달러 대출 환율 리스크
- [ ] 외국인 매수 시 必要서류 (印鑑証明·住民票 대체 → 영사관 발급 在留증명 등)

## F. 운영·관리 검증

- [ ] 임차인 모집 의뢰 가능한 PM社 후보 2~3개 가격 비교 (3-5% 임대료)
- [ ] 빈집 발생시 平均 空室기간 (도쿄 도심 1LDK = 약 2-4주)
- [ ] 단지 내 賃貸관리 規約 (短期賃貸 금지·airbnb 금지 등)
- [ ] 외국인 임차인 허용 정책
- [ ] 보증회사 의무가입 여부

## G. 세금·경비 검증

- [ ] 不動産取得税 (가격×3%×軽減율)
- [ ] 登録免許税 (소유권移轉 0.3-2%, 抵当権설정 0.4%)
- [ ] 印紙税 (계약가에 따라 ¥10k-60k)
- [ ] 仲介手数料 (3% + ¥60k + 消費税)
- [ ] 司法書士 보수 (¥80k-150k)
- [ ] 매년 固都税 (가격×1.4%, 軽減後 실효 0.15-0.30%)
- [ ] 임대수익에 대한 소득세 신고 의무 (확정신고 필요)
- [ ] 한국 거주자라면 한일조세조약 + 한국 종합과세 + 외국납부세액공제

## H. 매도·환금 시나리오

- [ ] 5년·10년 후 매도시 양도소득세 (5년 초과시 約 20%, 미만시 約 39%)
- [ ] 단지 내 過去 6개월 거래사례 3건+ (REINS 조회 - 중개사 통해)
- [ ] 賃料 vs 매도 시나리오의 비교 (NPV 둘 다 계산)

## I. 최종 풀(10건) 확정 기준

다음 모두 통과시에만 최종 10건 풀에 포함:

| 항목 | 기준 |
|---|---|
| A 매물 기본 | 모든 박스 체크 |
| B 빌딩 | 신내진 OR 보강 완료 + 수선적립금 적정 + 총세대수 ≥ 100 |
| D 재무 | DSCR ≥ 1.0x (LTV 70%, 2.5%, 임대료 -10% stress 시 ≥ 0.9x) |
| E 차주자격 | 사용자 본인의 차주 자격에서 실제 대출 실행 가능 |

## J. 실행 순서 요약

```
[1] research/tokyo_core_buildings.json 의 12개 빌딩 후보 검토
        ↓
[2] 일본 IP / VPN 환경에서 SUUMO·HOMES·athome 접속
        ↓
[3] 각 빌딩 페이지에서 현재 매도 호실 URL 수집 → urls.txt
        ↓
[4] cd scraper && npm install && npx playwright install chromium
[5] npm run scrape  →  out/<slug>/property.json + 이미지
        ↓
[6] property.json 들을 모아 sample_input.json 형태로 정리
[7] python3 src/underwrite.py --input data/listings.json
        ↓
[8] DSCR ≥ 1.0x 통과한 매물에 본 체크리스트 적용
        ↓
[9] 최종 10건 선정 → data/properties.json
        ↓
[10] python3 render/render.py --mode both --input data/properties.json --output output/
        ↓
[11] STARTS 매물카드 10장 + Executive Analysis PDF 1장 생성 완료
```

- 일본 거주자 신분이 아니라면 매수·매도·임대운영 전 과정의 세무사·행정서사 비용을 모델에 반영
